# C Programming Practicals

A collection of C programming exercises covering arrays, pointers, dynamic memory allocation, and string manipulation.

## 📋 Overview

This repository contains solutions to 10 practical problems designed to reinforce core C programming concepts, including:

- Array traversal and manipulation
- Pointer arithmetic
- Dynamic memory allocation (`malloc`/`new`) and deallocation
- String processing with pointers
- Searching, sorting, and merging algorithms

## 🗂️ Practicals

| # | Title | Concepts Covered |
|---|-------|-------------------|
| 1 | Count Even and Odd Numbers | Arrays, loops, conditionals |
| 2 | Linear Search | Arrays, function return values |
| 3 | Descending Range Array (Dynamic) | Dynamic memory allocation, error checking |
| 4 | Swap Middle Characters | Strings, function parameters |
| 5 | Modify Values Using Pointers | Pointers, pass-by-reference |
| 6 | Reverse Array (Dynamic + Pointers) | Dynamic memory, pointer arithmetic |
| 7 | Replace Vowels with `*` | Strings, pointer dereferencing |
| 8 | Merge and Sort Two Arrays | Dynamic memory, sorting algorithms |
| 9 | Swap First and Last Elements | Arrays, pointers |
| 10 | Print All Substrings of Length 3 | Strings, pointer traversal |

## 🛠️ Requirements

- A C compiler (GCC recommended)
- Terminal / command line access

## ▶️ How to Compile and Run

Each practical is contained in its own `.c` file. To compile and run:

```bash
gcc practical_01.c -o practical_01
./practical_01
```

Replace `practical_01` with the relevant file name for each exercise.
